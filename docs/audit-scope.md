# Audit Umfang (Scope)

## Im Umfang

- Windows 10 Enterprise (21H2+)
- Windows 11 Enterprise
- Lokale Sicherheitsrichtlinien
- Gruppenrichtlinienobjekte (GPO)
- Windows Defender Konfiguration
- Windows Firewall Konfiguration
- BitLocker Verschlüsselungsstatus

## Außerhalb des Umfangs

- Server-Betriebssysteme
- Netzwerk-Infrastruktur
- Cloud-Dienste
- Drittanbieter-EDR-Lösungen

## Annahmen

- Administratorrechte sind verfügbar
- Aktuelle kumulative Updates installiert
- Domänenrichtlinien gelten, wo anwendbar
