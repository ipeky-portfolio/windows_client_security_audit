# Windows 10/11 Sicherheits-Audit – Basierend auf CIS Benchmarks

## Überblick

Dieses Projekt zeigt ein strukturiertes Windows Client Sicherheits-Audit basierend auf den CIS Benchmarks des **Center for Internet Security (CIS)**.

Die Audit-Methodik folgt bewährten Branchenstandards und bewertet Windows 10 und Windows 11 Systeme anhand der Level 1 Sicherheits-Empfehlungen.

## Ziele

- Sicherheitskonfiguration von Windows-Clients prüfen
- Abweichungen von CIS Benchmark Empfehlungen identifizieren
- Risikobasierte Ergebnisse liefern
- Konkrete Handlungsempfehlungen geben
- Professionelle Audit-Methodik demonstrieren

## Umfang

Zielsysteme:
- Windows 10 Enterprise
- Windows 11 Enterprise
- Domänen-gebundene Clients
- Standalone Systeme (optional)

CIS-Profil:
- Level 1 (Unternehmensstandard)
- Level 2 (optional – hohe Sicherheit)

## Methodik

Das Audit folgt einem 5-Phasen Ansatz:

1. Systemidentifikation
2. Konfigurations-Extraktion
3. CIS Control Validierung
4. Risikobewertung
5. Reporting & Remediation-Planung
